# HIGO Automation Testing (Selenium)

## Tools
- Selenium
- Python
- Chrome WebDriver

## Websites
- https://higo.id
- https://blog.higo.id

## Test Scenarios
- test_homepage_load_successfully
- test_navigation_to_blog
- test_career_menu_redirect_to_linkedinx
- test_blog_homepage_load
- test_open_latest_blog_article
- test_blog_search_valid_keyword

## How to Run
1. Install dependency
   pip install -r requirements.txt
2. Run test
   python tests/test_higo.py
   python tests/test_blog.py
